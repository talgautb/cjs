# Template

What in template:

- [pug](https://pugjs.org/)
- [stylus](http://stylus-lang.com/)
- JavaScript
- gulp (NPM)

# How to start

```
npm i -g gulp babel
npm i
```

## Commands

**Start develop:**

- `npm start` or `gulp start` (localhost:4001 - *default*)

**Build project:**

- `gulp build`

**Create zip file from /dist/**

- `gulp zip`
